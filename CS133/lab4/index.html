Creating a LaTeX document

  1. The first step in creating a LaTeX document is to declare the documentclass and subsequent preamble. The documentclass declaration specifies the type of LaTeX document that should be made; e.g. article, book, etc. The preamble sets up the LaTeX environment by setting document properties and also allows for packages to be imported.

    \documentclass{article} % Uses the article document class.

    %Preamble start
    \title{My First LaTeX Document}
    \author{Aaron Baw}
    \date{\today}
    % END of preamble.

    After declaring the preamble, the main section of the document can be created.

    \begin{document}
    \maketitle % Prints heading information e.g. title
    % DOCUMENT CONTENT HERE...
    \end{document}

  2. After setting up the document, I compiled it from the terminal using pdflatex:

    pdflatex firstDoc.tex

Sectioning Content

Depending on the LaTeX document type (declared in the document type), it may be appropriate to section off the content to make it more organized and readable.

  1. In LaTeX, there are many different sectioning options, such as by chapter, section and subsections. I tried out a few of these in my document.

    \chapter{The Beginning, of the beginning.}
    Sub-section upon sub-section upon sub-section upon sub-section...
    \section{The First section, of the beginning, of the beginning.}
    \subsection{The first sub-section of the first section of the beginning of the beginning.}

  2. There are a couple more useful commands to organize documents such as \tableofcontents and \appendix which conveniently generate content tables and appendices.

    \tableofcontents
    \appendix

  3. A curious thing to note about LaTeX is that the usage of quotation marks is slightly diffeerent to what one would expect. Instead of using regular appostrophes, it is required to use a pair of back ticks for the start of the quote, and a pair of single apostraphes at the end of the quote.

    ``This is how you quote in LaTeX''

Laying out Content

Different environments can be declared inside a LaTeX document to display information in different ways.

  1. One example of such environments is enumeration in LaTeX which is a type of numbered list that I used in my document.

    \begin{enumerate}
    \item1 Content...
    \item2 More content...
    \end{enumerate}

Embedding figures / images

    Displaying figures utilizes a type of environment such as those outlined above.

  1. An additional package graphicx is required in order to embed figures with LaTeX.

    \usepackage{graphicx}

    This line is added within the preamble.

  2. To embed an image I set up the environment and added the image as follows:

    \begin{figure}[h]
    \begin{center}
    \includegraphics[width=80mm]{/dcs/16/u1617781/Pictures/sample-cat.jpg}
    \caption{An adorable cat}
    \label{reflabel}
    \end{center}
    \end{figure}

    There are several options that can be altered when using the figure environment in LaTeX, such as [width=x] or [height=x] which allows you to customize the desired height/width of the embedded image.

    The [h] selector tells the environment where to most appropriately place the figure, in this case in relation to the header.

Mathematical Equations and other symbols

  1. Mathematical equation environments can be used in LaTeX to display equations, such as the quadratic formula which I used in the following example:

    \begin{equation}
    x=\frac{-b \pm \sqrt{b^{2} - 4ac}}{2a}
    \label{quadraticFormula} \end{equation}

    The \frac{...}{...} syntax tells latex to treat the content in each of the curly braces as the numerator and denominator of a fraciton. \pm embeds a plus or minus symbol, and I also used \sqrt{...} to wrap the square root portion of the formula within the appropriate symbol.

  2. Another useful capability of using figures is that they can be defined by using \label{name} and then referenced later to be embedded as a copy. The benefit to this is that if the original content changes, the rest of the figures update to reflect this change.
