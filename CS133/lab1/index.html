Commands:

  • cp: Copy a file. Requires two arguments, source and destination. An example might be

    cp ~nick/cs133/infotutorial ./infotutorial

  • chmod: Edit the permissions for a file or directory.

    chmod -R a+r
    chmod o+x

    Requires that you specify the Group type (Owner, All, User, Group) as well as permission (Read, Write, Execute denoted by rwx), and also typically whether this is added or removed.

  • ssh: Remotely access the machine from a terminal.

    ssh username@server

  • ls -l: Returns a list of files and other useful information, such as the permissions for each file / directory, size in bytes, date of last modification and the owner.

  • pwd: Prints the working directory.

    pwd
